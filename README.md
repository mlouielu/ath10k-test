ath10k test
===========

This repo contain the test data for ath10k driver optimization.

Using `flent` to make the test cases.

TEST
====

* rtt_fair_up_cs0
* rtt_fair_up_cs7
* udp_flood_up_cs0
* udp_flood_up_cs7

HOW-TO
======

## Install flent

```
$ sudo python3 -m pip install flent
```

## Open flent GUI

```
$ flent --gui
```

Then use gui to open `.flent.gz` file to see the plots
